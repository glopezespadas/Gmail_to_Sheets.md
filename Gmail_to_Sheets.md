# Gmail to Google Sheets Automation

## Overview
Real-time email capture system that automatically logs incoming Gmail messages to Google Sheets with structured data organization.

## What It Does
Captures incoming emails and stores them in Google Sheets with:
- **From**: Sender email address
- **Subject**: Email subject line
- **Date**: Timestamp of receipt
- **Body**: Email content

## How It Works
1. Gmail receives email (trigger)
2. Make.com detects the new message
3. Automatically adds row to Google Sheets
4. Data is organized and timestamped

## Results
✅ Automated email capture in real-time
✅ Zero manual data entry
✅ Organized email repository

## Tech Stack
- **Automation**: Make.com
- **Email**: Gmail API
- **Database**: Google Sheets
- **Integration**: REST APIs
