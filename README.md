# Gmail to Google Sheets Automation

## Qué hace
Cada email que llega a Gmail se guarda automáticamente en Google Sheets con:
- From (quién envía)
- Subject (asunto)
- Date (fecha)
- Body (contenido)

## Cómo funciona
1. Gmail recibe email (trigger)
2. Make.com lo detecta
3. Agrega fila en Google Sheets
4. Datos quedan organizados

## Resultado
Sistema automático funcionando. Los datos se capturan en tiempo real.
